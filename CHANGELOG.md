# Nouveautés v19 — Le Chat 💬

Le chaînon manquant : UNE conversation pour tous tes cerveaux.

- **Nouvel onglet 💬** : vraies bulles de discussion (toi à droite en bleu,
  l'IA à gauche), historique **sauvegardé par profil**, micro 🎤, réponses à
  voix haute 🔊, bouton 🗑 pour repartir de zéro.
- **Choix du répondant** :
  - 🤖 **Auto** : le plus malin disponible — distant > enfant > code local.
  - 💻 Cerveau code local (tes complétions, réglées par la créativité du profil).
  - 👶 Ton enfant actif — discuter ici le fait grandir !
  - ☁️ Cerveau distant.
- **Contexte intelligent** : le cerveau distant reçoit l'état de tes IA
  (étiquettes connues, dernier objet vu, dernier son entendu, profil actif) —
  ses réponses parlent vraiment de TON app : « je vois que tu as appris
  12 photos de chats aujourd'hui... »
- **Discuter = entraîner** : chaque message que tu écris nourrit l'IA code.
- La liste des répondants se met à jour selon tes enfants et l'état du
  cerveau distant.
